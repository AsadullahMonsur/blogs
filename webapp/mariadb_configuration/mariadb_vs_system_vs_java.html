Version Validation Notes

  • Ubuntu 22.04 default JDK remains OpenJDK 11.
  • Ubuntu 24.04 default JDK is OpenJDK 21.
  • MariaDB Connector/J 3.x supports modern Java LTS versions: Java 8 → Java 25.
  • MariaDB 10.11 is the current long-term stable branch for enterprise deployments.
  • MariaDB 11.x is recommended for new systems using Ubuntu 24.04 and Java 21+.
  • Java 21 is currently the preferred production LTS runtime for modern deployments.
