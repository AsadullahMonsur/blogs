Installing MariaDB 10.11.15 from Source

Portable local installation on Ubuntu without system-wide installation.

Official Documentation

MariaDB official build documentation:

https://mariadb.com/docs/server/server-management/install-and-upgrade-mariadb/installing-mariadb/compiling-mariadb-from-source/building-mariadb-on-ubuntu/

1. Target Directory Layout

/home/cinnamon/softwares/mariadb-10.11.15   (source)
/home/cinnamon/softwares/mariadb           (final install)
/home/cinnamon/softwares/mariadb/data      (database files)
/home/cinnamon/softwares/mariadb-build     (temporary build directory)

3. Download & Extract MariaDB Source

cd /home/cinnamon/softwares

wget https://archive.mariadb.org/mariadb-10.11.15/source/mariadb-10.11.15.tar.gz

tar -xzf mariadb-10.11.15.tar.gz

4. Create Required Directories

mkdir -p /home/cinnamon/softwares/mariadb
mkdir -p /home/cinnamon/softwares/mariadb/data
mkdir -p /home/cinnamon/softwares/mariadb-build

5. Configure Build (CMake)

cd /home/cinnamon/softwares/mariadb-build

cmake /home/cinnamon/softwares/mariadb-10.11.15 \
  -DCMAKE_INSTALL_PREFIX=/home/cinnamon/softwares/mariadb \
  -DMYSQL_DATADIR=/home/cinnamon/softwares/mariadb/data \
  -DWITHOUT_TOKUDB=1 \
  -DWITHOUT_ROCKSDB=1 \
  -DWITH_SSL=system \
  -DWITH_ZLIB=system \
  -DWITH_UNIT_TESTS=0
This config creates a portable local MariaDB installation.

6. Compile MariaDB

Recommended for normal systems:

make -j$(nproc)

For low memory systems:

make -j1

7. Install MariaDB

make install
MariaDB platform binaries will be available into:

/home/cinnamon/softwares/mariadb

now we can setup mariadb portable way by following instructions

8. Create Configuration Exporter Script

Create configuration_exporter.sh: 

nano /home/cinnamon/softwares/mariadb/configuration_exporter.sh

Paste the following script: 

#!/usr/bin/env bash

# Fixed MariaDB port
PORT=3307

# Current directory = MariaDB root folder
ROOT_DIR="$(pwd)"

# Create required directories if missing
mkdir -p "$ROOT_DIR/tmp"
mkdir -p "$ROOT_DIR/logs"
mkdir -p "$ROOT_DIR/data"

# Generate my.cnf
cat > "$ROOT_DIR/my.cnf" <<EOF
[client]
port=$PORT
socket=$ROOT_DIR/tmp/mysql.sock

[mysqld]
port=$PORT
socket=$ROOT_DIR/tmp/mysql.sock
basedir=$ROOT_DIR
datadir=$ROOT_DIR/data
tmpdir=$ROOT_DIR/tmp
pid-file=$ROOT_DIR/tmp/mysqld.pid
log-error=$ROOT_DIR/logs/mariadb-error.log
skip-networking=0
skip-grant-tables=0
skip-symbolic-links
EOF

echo "my.cnf generated successfully"
echo "Location: $ROOT_DIR/my.cnf"
echo "Port: $PORT"

Make the script executable: 

chmod +x /home/cinnamon/softwares/mariadb/configuration_exporter.sh

Run the exporter: 

cd /home/cinnamon/softwares/mariadb

./configuration_exporter.sh

9. Initialize Database

cd /home/cinnamon/softwares/mariadb

./scripts/mariadb-install-db \
  --basedir=$(pwd) \
  --datadir=$(pwd)/data \
  --user=$USER \
  --defaults-file=$(pwd)/my.cnf

10. Create Database Stopper Script

Create database_stopper.sh: 

nano /home/cinnamon/softwares/mariadb/database_stopper.sh

Paste the following script: 

#!/bin/bash

# Current directory = MariaDB root folder
ROOT_DIR="$(pwd)"

PID=$(pgrep -f "$ROOT_DIR/bin/mariadbd")

if [ -n "$PID" ]; then
    echo "Stopping MariaDB (PID: $PID)..."
    kill $PID

    # wait for it to stop
    sleep 15

    # force kill if still running
    if ps -p $PID > /dev/null; then
        echo "Force killing..."
        kill -9 $PID
    fi
else
    echo "No MariaDB process found."
fi

# stop any past instances
killall "$ROOT_DIR/bin/mysqld"
killall "$ROOT_DIR/bin/mariadbd"
pkill -f mariadbd

exit 0

Make the script executable: 

chmod +x /home/cinnamon/softwares/mariadb/database_stopper.sh

11. Create Database Starter Script

Create database_starter.sh: 

nano /home/cinnamon/softwares/mariadb/database_starter.sh

Paste the following script: 

#!/bin/bash

# Current directory = MariaDB root folder
ROOT_DIR="$(pwd)"

# Path to the stopper script
STOPPER_SCRIPT="$ROOT_DIR/database_stopper.sh"

# MariaDB start command
MYSQLD_CMD="$ROOT_DIR/bin/mysqld"
DEFAULTS_FILE="$ROOT_DIR/my.cnf"

echo "Stopping existing database service..."

# Execute stopper script
bash "$STOPPER_SCRIPT"

# Wait a few seconds to ensure shutdown completes
sleep 3

echo "Starting MariaDB..."

# Start MariaDB in background
"$MYSQLD_CMD" --defaults-file="$DEFAULTS_FILE" &

# Optional: save PID
MYSQL_PID=$!

echo "MariaDB started with PID: $MYSQL_PID"

Make the script executable: 

chmod +x /home/cinnamon/softwares/mariadb/database_starter.sh

12. Start MariaDB Using Script

cd /home/cinnamon/softwares/mariadb

./database_starter.sh

13. Stop MariaDB Using Script

cd /home/cinnamon/softwares/mariadb

./database_stopper.sh
