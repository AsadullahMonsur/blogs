REST vs GraphQL vs gRPC
Feature REST GraphQL gRPC
Protocol HTTP/1.1 HTTP/1.1 or HTTP/2 HTTP/2
Data Format JSON (commonly) JSON Protocol Buffers (binary)
Flexibility Fixed endpoints Flexible queries Strict contracts
Performance Moderate Efficient (reduces over-fetching) High (binary + HTTP/2)
Over/Under Fetching Common issue Minimal Minimal
Streaming Limited Limited (subscriptions) Built-in streaming
Ease of Use Easy Moderate learning curve Complex setup
Tooling Very mature Growing ecosystem Strong for microservices
Best For Public APIs, CRUD apps Frontend-heavy apps Internal microservices
Typing Loosely typed Strong schema Strongly typed (Proto)
