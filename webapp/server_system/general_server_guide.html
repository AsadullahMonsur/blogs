Server System Architecture

To check server system architecture:

uname -m

Enable a service of a user auto start at (boot up)

For user potato:

sudo loginctl enable-linger potato

Server Details

Username: potato
Server IP: 160.250.222.211
Destination: /home/potato/

Copy File to Server (SCP)

Copy mango.png to server in folder 'data':

scp mango.png potato@160.250.222.211:/home/potato/data

Copy Folder to Server (SCP)

Copy folder data to server:

scp -r data potato@160.250.222.211:/home/potato
